Primeiros passos

Estrutura

A estrutura de um arquivo html é composta 3 partes, html, head(cabeça) e body(corpo) .

Tags

As Tags são descrições que o navegador interpreta identificando o tipo de elemento inserido. Geralmente um elemento é composto por tag de abertura, elemento (texto) e tag de fechamento.

Principais tags:

  • h1 - Titulo;
  • h2 - Subtitulo;
  • h3 - Subtitulo de inportância menor que o h2;
  • p - Parágrafos.
